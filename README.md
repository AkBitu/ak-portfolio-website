# Ak Bitu Portfolio Website

## About This Project

This is my portfolio website created for my Web UI Design module assignment.

I built this website using Svelte to show my artwork, skills, experience, and contact information. The design was based on my earlier wireframes and prototypes in prototype module.

The website includes:

- About Me page
- Traditional Portfolio
- Digital Portfolio
- CV page
- Contact page

# Features

This website includes:

- Responsive design for mobile and desktop
- Hamburger menu for small screens
- Hover effects on buttons and icons
- Contact form with success message
- Keyboard focus for accessibility
- Reusable components like Header, Navigation, and Footer


# Technologies Used

- Svelte
- HTML
- CSS
- JavaScript
- GitHub


# Accessibility

To make the website more accessible, I added:

- Alt text for images
- Keyboard navigation focus
- HTML structure
- Responsive layouts
- Clear navigation menu
